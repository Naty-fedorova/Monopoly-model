# Monopoly-model
A simulation model of fission fusion dynamics and settlement formation, created as an exercise to remodel (and perhaps in the future expand) Enrico Crema's 2013 paper: Crema, E. (2014). A Simulation Model of Fission-Fusion Dynamics and Long-Term Settlement Change. Journal of Archaeological Method and Theory, 21(2), 385-404. Retrieved from http://www.jstor.org/stable/43655012
